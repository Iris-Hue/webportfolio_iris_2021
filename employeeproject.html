Employee Directory Project

This project used an API (Application Programming Interface) to request the 12 random users using the fetch method. A third-party web server stores the data, then that data needs to be returned in JSON format from the server to show the data on the page.

Challenges and Achievements

I learned how to look up a CDN (content delivery network) plugin to improve loading time for JavaScript and CSS files on the modal window. I'm starting to feel how HTML and CSS work. The CSS styling was neat when I used z-index for the modal window to lay on top of the body using absolute positioning. And the relative position on the body with a semi-transparent background.

Conclusion

This project was nearing the end of the TreeHouse tech degree. At this point, I understood code semantics which I had fun working on this assignment. I started by building a static page with HTML and CSS for the body. I used CSS grid to layout the employee cards. By using slack and rewatched Treehouse videos and references, I was able to complete this with ease.
