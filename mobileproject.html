Mobile-First Profile Project

This responsive layout was my first project to create from scratch and scale to multiple screen sizes. HTML to create the content and the basic structure was header main and footer. CSS to style the page for a mobile view first. Then media queries breakpoints to adjust the width of the screen for larger screens.

Challenges and Achievements

I had to overcome old fears of downloading other apps that corresponded to a repository on Github is one example. And I learned how folder structure was created with the parent folder and then the secondary folder. This demanding project, but the hard work that went into it and the vast learning curve have their payoffs. I learned flexbox, which I used on the photo gallery. I learned the flexbox one-dimensional concept for the layout and spacing of the gallery. Also, I learned to use chrome Dev Tools which is a great asset for making small changes and seeing those right away.

Conclusion

Being the second project in Treehouse tech degree, there were many new concepts to learn. It was overwhelming but, reaching out to the Instructor and the chat community/support on Slack made it possible to complete this assignment with an exceeds expectations grade.
